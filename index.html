Frequently Asked Questions

How does Luaxperience work?

Luaxperience creates a specialized runtime environment for Roblox, bridging the gap between Android and Linux, allowing native-like performance on non-Windows platforms.

Why can’t I see my app bundle?

If you’re experiencing issues with the file picker, it may be due to missing Desktop Portals on your system. Check out this guide for more info.

Why does Luaxperience keep crashing?

Luaxperience is still in experimental stages, so crashes are expected. Help us improve by submitting logs through the terminal using the command flatpak run org.vinegarhq.Sober.

Is Luaxperience official?

No, Luaxperience is a community-built project. While it’s not affiliated with Roblox, it’s developed with the best practices to ensure safety and performance.
