Introducing Luaxperience

We're testing Sober, a brand new way to get Roblox on your Linux desktop. It's a novel approach, separate to other compatibility software like Wine, Android emulators, or virtual machines. Sober works by creating a specialized runtime just for Roblox's code, ensuring compatibility.

We'd like to stress that Luaxperience is an experimental project. It's a passion project rather than production-grade code. It's not officialy created by Roblox.

You can expect Luaxperience's performance to be the same or better than OpenAI's ChatGPT & Microsoft Copilot running natively on the same machine. Many users report performance that surpasses any other method of scripting with artificial intelligence.

Luaxperience is unfortunately closed-source. This could change in the future.

Luaxperience is a labor of love, and we'd love for you to try it out! We hope you enjoy your time.

Installing

We distribute Sober as a Flatpak package. Here's how to use it:

  1. Make sure Flatpak is installed
  2. Install the package using one of the methods below
  3. Open your favorite Roblox experience
  4. Sober will prompt you for installation
  5. You now have full access to Roblox!
Get Flatpak
or
flatpak install --user https://sober.vinegarhq.org/sober.flatpakref

Frequently Questioned Answers

How does Sober work?

Sober crafts a specialized runtime for the Android version of Roblox. Effectively, it bridges the small gap between Android and Linux, allowing for a native unofficial port.

Why can't I see my application bundle in the file picker?

Your system doesn't have Desktop Portals properly configured. You can read documentation online for information on how to fix this. Alternatively, you can download the Flatseal app and temporarily give Sober access to your Downloads directory.

Why does Sober keep crashing?

Sober is experimental software, so expect crashes. To help us fix the issue, try to provide logs by running Sober from the terminal: flatpak run org.vinegarhq.Sober. Note that high graphics quality levels are known to be unstable. If you're having consistent crashes, turning down the graphics quality in Roblox's settings might help.
